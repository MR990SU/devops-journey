# AWS IAM Basics

## Concepts

- User
- Group
- Policy
- Least Privilege

## Commands Used

- aws iam create-user
- aws iam create-group
- aws iam attach-group-policy
- aws iam add-user-to-group
- aws iam list-policies
## Learning

- IAM controls access securely
- Avoid AdministratorAccess in production
- Use least privilege principle

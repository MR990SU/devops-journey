# AWS CLI Basics

## Commands Used

- aws configure
- aws sts get-caller-identity
- aws s3 ls  >> aws s3 ls s3://subhajit-devops-bucket-001/
- aws s3 mb  >> aws s3 mb s3://subhajit-devops-bucket-001
- aws s3 cp  >> aws s3 cp test.txt s3://subhajit-devops-bucket-001/

## Outcome

- Configured AWS CLI
- Created S3 bucket
- Uploaded file to S3

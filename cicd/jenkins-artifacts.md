# Jenkins Environment Variables and Artifacts

## What I did

- Created a Jenkins freestyle job
- Used Jenkins environment variables in build steps
- Generated a build artifact
- Archived artifact in Jenkins

## Important Variables Used

- JOB_NAME
- BUILD_NUMBER
- WORKSPACE
- NODE_NAME

## Artifact Created

build-report.txt

## Important Concepts

- Workspace is where Jenkins runs the build
- Artifact is the important output preserved after build
